# Hamburger Menu

Este proyecto implementa un menú hamburguesa interactivo utilizando **HTML**, **CSS** y **JavaScript**.  
El menú se expande y colapsa al hacer clic en el ícono hamburguesa, mostrando opciones de navegación de manera elegante y accesible.

---

## 🚀 Características
- Diseño minimalista y moderno.
- Animación suave al expandir/colapsar el menú.
- Opciones de navegación: Home, Blog, About, Teams, Contact.
- Accesibilidad mejorada con atributos `aria-expanded` y `aria-label`.
- Código modular y fácil de personalizar.

---

## 📂 Estructura del proyecto
Menu-Hamburguesa-simple/

│── style.css    # Estilos del menú
│── menu.js    # Lógica de interacción

## 🛠️ Tecnologías utilizadas
- **HTML5**: estructura semántica.
- **CSS3**: estilos, transiciones y efectos hover.
- **JavaScript (ES6)**: control de eventos y manipulación del DOM.
